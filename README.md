# SpeakUP

This reppo is the backend for SpeakUP MERN website.


## Technologies used

Build with : 

- MongoDB
- Express
- NodeJS
- Socket.io

Hosted on :
- Render https://speakup-server.onrender.com/

## Setup and usage : 
- Clone the repository
- npm install
- Install the dependencies from package.json
